一、翻译文档格式要求：

    1、文档开头部分必须要用“//”来表名当前文档的名称。

    2、每行文本之间必须要有换行，除了一些包裹符号。

    3、大的内容必须用“一、二、三、四”等标识。

    4、分点的内容用"1、2、3、4"等标识。

    5、大点和小点之间必须要有缩进，以保证美观。

    6、每行文本的字数不要太多，最好不要超过80个。

    7、在对函数进行描述的时候，函数前面要加“@”符号，并且符号后面加空格，然后在函数后面加冒号。

    8、对于一些专业名词，不要翻译，例如Activity就不用翻译了。

    9、对于要很详细的描述的一些点，用"*"把点的名称包裹起来。

二、注意

    1、本项目是在WebStorm编辑器下创建的项目，所以建议用该编辑器打开整个目录。
